# shipstation-api
[![Build Status](https://travis-ci.org/hurdad/shipstation-api.svg?branch=master)](https://travis-ci.org/hurdad/shipstation-api)

Java 8 interface to ShipStation HTTP API

## Build Requirements
```
Java 8 JDK
Maven
```

## Build
```
mvn
```

## Run Test
```
mvn test
```

## Create JAR
```
mvn package
```
